JSON
====

The JSON extension provides the 'json' Output Handler, as well as the '--json'
command line option.  Any command functions that use the 'render()' function
will output the rendered dictionary in the JSON format.  This extension is
only supported on Python 2.6+, unfortunately because jsonpickle does not yet
support Python 3.
